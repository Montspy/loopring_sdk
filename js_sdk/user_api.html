Customer KeySeed format

BaseAPI.KEY_MESSAGE is a static for BaseAPI, storage loopring Sign message with key nonce: ${nonce} should be end of the string

_**Preview loopring customer keySeed is default value, Api will return an empty keySeed, please keep them signature follow the loopring message

BaseAPI.KEY_MESSAGE.replace("${exchangeAddress}",exchangeInfo.exchangeAddress).replace("${nonce}", (nonce - 1).toString())**_

More detail read unit test updateAccount & getUserApiKey & customer_keySeed
